# Rairaiken Delivery App

## Windows and Linux 

Download the latest version of the Rairaiken Delivery App from the [releases page](https://github.com/JojosBizarreCoding/rairaiken-delivery-app/releases/latest)

## MacOS

For MacOS, build/run the app yourself. See the instructions below.

## Building and running
1. Make sure you have [Node.js](https://nodejs.org/en/download/) installed.
2. Navigate to the `electron` folder in your terminal.
3. Run `npm install` to install dependencies.
4. Run `npm start` to launch the app in development mode.
5. To create a packaged version of the app, run `npm run app:dist`. This will generate installers in the `electron/dist` folder.
